---- React Application Setup Steps ----
1- npm create vite@latest my-vue-app
2- go inside folder name ans install - npm i 
3- install tailwind - npm install tailwindcss @tailwindcss/vite
4- vite.config.ts set up --
		import { defineConfig } from 'vite'
		import tailwindcss from '@tailwindcss/vite'
		export default defineConfig({
		plugins: [
			tailwindcss(),
		],
		})

5- intall all state management Redux and Redux-toolkit 
		npm install react-redux
		npm install @reduxjs/toolkit
